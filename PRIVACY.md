# MoOps 隐私与数据流说明

MoOps 是一款**单机桌面应用**，所有运营数据均保存在你本地机器，不上传到任何 MoOps 服务器。

## 数据存储位置
- 应用状态（设置、各模块台账、品牌库等）保存在本机 Electron 的 `localStorage`（键名 `ragc_v2`），位于用户数据目录，不离开本机。
- 你可以通过菜单「导出数据 / 导入数据」将全部数据备份为单个 JSON 文件，或导入恢复。

## 对外网络请求
应用只在以下情况发起对外请求，且均由你主动触发：

1. **AI 生成 / 识图**：请求发往你在设置中配置的 AI Provider（OpenAI / DeepSeek / 小米 MIMO / Claude / 自定义）。请求内容包含你输入的需求文本与（识图时）图片数据；API Key 仅存储于本机并随请求发送，不经 MoOps 中转。
2. **合规远程规则库**：合规检测会从 GitHub（JSDelivr CDN）拉取 `compliance-rules.json` 作为易变平台规则补充，仅含规则文本，不含你的任何内容。
3. **网页抓取（可选）**：粘贴链接抓取正文时，由主进程代理请求该网页，仅用于本地解析展示。
4. **更新检查**：启动时 / 手动点击「检查更新」时读取 GitHub 上的 `version.json`（含版本号、下载地址、SHA-256），不发送任何个人信息。

## 我们不会做的事
- 不收集、不上传你的运营数据、内容草稿、API Key。
- 不内置遥测 / 行为追踪。
- 不向第三方共享任何数据。

## API Key 安全说明
当前版本 API Key 以明文保存在本机 `localStorage`。请妥善保管本机，并善用「导出 / 导入数据」做备份。后续版本评估引入本地加密存储。
